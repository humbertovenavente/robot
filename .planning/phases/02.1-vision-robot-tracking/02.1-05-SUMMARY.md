### Key Assertions

**Test A (enabled happy path):** Patches `find_robot_qr` to return `(321, 242)` against home target `(320, 240)` → drift ≈ 2 px. Reads back the tmp_path JSONL file and asserts `vision_confirmed=True`, `drift_px≤30`, `vision_reason="ok"`.

**Test B (drift_exceeded):** Patches `find_robot_qr` to return `(500, 500)` against home target `(320, 240)` → drift > 30. Asserts `vision_confirmed=False`, `vision_reason="drift_exceeded"`, cycle still `"completed"`, `robot.deposit` and `robot.return_home` each called once, `_cycle_lock` is `False`.

**Test C (disabled path byte-identical):** Asserts `find_robot_qr.call_count == 0`, then reads the JSONL line and compares its frozen key-set against `PHASE1_SCHEMA_KEYS = {timestamp, class, destination_bin, cycle_time_ms, status, error}`. Explicitly asserts `"vision_confirmed" not in record`, `"drift_px" not in record`, `"vision_reason" not in record`.

**Test D (VIS-06 YAML handoff):** Writes a full `station_config.yaml` via `yaml.safe_dump`, calls `write_pixel_target_to_yaml` (Plan 03 helper) to persist `home=(320,240)`, then calls `load_config(str(yaml_path))` to get a real `Config` object. Asserts `cfg.robot_vision_targets["home"] == [320, 240]`. Runs a cycle with `find_robot_qr` returning the exact calibrated target → asserts `drift_px == 0`.

**Test E (freeze guard):** Reads `station.py` source and asserts `"from orchestrator"`, `"from ws_protocol"`, `"from templates"`, and `"from dashboard"` are absent.

**Test F (REQ-ID audit):** Reads `Path(__file__).read_text()` and asserts all four tokens `"VIS-05"`, `"VIS-06"`, `"VIS-07"`, `"VIS-08"` appear. Final count: 39 occurrences.

## Opt-in Posture

`vision_confirm_enabled: false` is the YAML default. The operator must:
1. Run `python calibrate.py` with a connected camera to capture pixel targets.
2. Set `vision_confirm_enabled: true` in `station_config.yaml`.
3. Restart `station.py`.

When disabled, pyzbar is never invoked post-motion and the JSONL schema is Phase 1 v1.0.
